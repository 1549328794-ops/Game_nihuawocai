# 项目任务列表

## 项目概述
创建一个在线你画我猜游戏，玩家可以在画布上作画，AI 系统负责猜测画的是什么内容。

## 技术栈
- 脚手架: Vite
- 前端: React + TypeScript
- 画布: HTML5 Canvas
- 后端: Node.js + Express
- AI 集成: OpenAI Vision API
